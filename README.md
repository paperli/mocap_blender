# MoCáp Addon in Blender
Blende Add-on for real time motion capturing through iPhone/iPad camera. Work with MoCáp iOS app.

## Install
1. Download the project
2. Open the enclosed `example.blend` file in Blender
3. In Blender, go to `Edit` -> `Preferences` -> `Add-ons` -> `Install...` and select `Server.py` to install the add-on
4. In `Object Mode`, open side panel by press N key in keyboard, and select `MoCap` tab
5. In the `MoCap Server` panel, assign `biped_robot_ace_skeleton` in `Skeleton` field.
6. Press `Start Server` button to start the MoCáp server
7. Open MoCáping app in iPhone
8. Enter MoCáp server's IP address (It shall display in `MoCap Server` panel in Blender)
9. Voilà! Let's dance!
